# BGM 预置音乐库

将以下免版权 MP3 文件放在此目录，系统将自动提供预置 BGM 选择：

| 文件名              | 风格说明                         |
|---------------------|----------------------------------|
| bgm_energetic.mp3   | 活力节拍，适合运动/时尚品类      |
| bgm_calm.mp3        | 轻柔简约，适合家居/美妆/食品品类 |
| bgm_corporate.mp3   | 商务大气，适合数码/奢侈品品类    |
| bgm_trendy.mp3      | 潮流时尚，适合服装/潮玩品类      |

## 推荐免版权音乐来源

- [Pixabay Music](https://pixabay.com/music/) — 下载后重命名放入本目录
- [Free Music Archive](https://freemusicarchive.org/)
- [YouTube Audio Library](https://studio.youtube.com/channel/UC/music)

## 注意

文件不存在时，前端会将对应预置项显示为灰色不可用状态。用户仍可上传自定义 BGM。
