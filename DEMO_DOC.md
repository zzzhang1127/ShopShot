# ShopShot — AI 电商带货视频一键生成平台
## 项目演示文档

---

## 一、项目背景与简介

### 1.1 背景

随着抖音、快手、小红书等短视频平台的爆发式增长，电商卖家对**带货短视频**的需求日益旺盛。然而，传统的视频制作流程繁琐：需要撰写文案、设计分镜、拍摄/剪辑视频、配音配乐，对中小商家而言成本高、效率低。

**ShopShot** 正是为解决这一痛点而生——让商家只需**上传商品图片 + 一句话描述**，AI 即可在几分钟内自动完成从剧本到成片的全流程创作。

### 1.2 项目简介

**ShopShot** 是一款 AI 驱动的电商带货视频自动生成平台，核心能力包括：

- 🎬 基于 AIDA 营销模型，AI 自动生成带货剧本与 4 镜分镜方案
- 🎥 调用 Seedance-1.5-pro 视频大模型，逐镜生成高质量短视频
- 🗣️ 集成 TTS 语音合成，自动为每个分镜配上电商风格人声
- 🎵 智能 BGM 匹配，支持预设曲库或自定义上传背景音乐
- 🔄 全自动后处理流水线，完成拼接、配音、混音、时长适配

> **核心价值：让一名没有视频制作经验的普通商家，也能在 5 分钟内产出一条专业级带货短视频。**

## 二、功能介绍

### 2.1 工作台总览

ShopShot 采用 **4 步线性工作台**设计，引导用户从商品信息到成品视频：

```
【板块1】商品输入 → 【板块2】剧本编辑 → 【板块3】分镜配置 → 【板块4】视频呈现
```

> 📸 **截图提示**：截取项目详情页的整体工作台视图（4 个板块从上到下展示的完整页面）

---

### 2.2 板块 1：商品信息与剧本生成

**功能描述：**
- 上传 1-4 张商品图片（支持拖拽/URL 导入/本地上传）
- 输入商品名称与描述
- 点击「生成剧本」→ AI（Seed-2.0-pro 多模态大模型）实时分析商品图片，**流式输出**带货文案

**技术亮点：**
- 采用 SSE（Server-Sent Events）流式响应，文字逐字实时出现，体验更流畅
- 系统 Prompt 基于电商文案五段式结构：开场钩子 → 痛点 → 卖点 → 情感共鸣 → 行动引导

> 📸 **截图提示 1**：截取板块1 区域，包含已上传的商品图片和剧本文本框（最好有正在流式输出文字的状态）

---

### 2.3 板块 2：剧本编辑与分镜生成

**功能描述：**
- 可对 AI 生成的剧本进行自由编辑修改
- 设置视频时长（5 / 10 / 15 / 20 秒）和画面比例（9:16 / 16:9 / 1:1）
- 支持上传参考视频作为镜头风格参考（自动提取运镜风格）
- 点击「生成分镜提示词」→ AI 基于 AIDA 模型生成 4 个分镜的图像描述和运镜方案

**AIDA 营销模型：**
| 分镜 | 对应 | 作用 |
|------|------|------|
| P1 | Attention（注意） | 开场吸睛画面，抓住用户视线 |
| P2 | Interest（兴趣） | 展示商品特色，激发好奇心 |
| P3 | Desire（欲望） | 强化购买欲，展示使用场景 |
| P4 | Action（行动） | 引导下单的行动号召 |

> 📸 **截图提示 2**：截取板块2 区域，包含剧本文本框和分镜模板上传区域

---

### 2.4 板块 3：分镜提示词配置与音频设置

**功能描述：**
- 展示 4 个分镜的图像提示词和动作描述，支持逐镜手动编辑
- 配置 TTS 人声：可选择 5 种中文神经网络音色（小晓/云希/晓伊/云健/晓墨）
- BGM 设置：4 种预设曲库风格（活力节拍/轻柔简约/商务大气/潮流时尚）或自定义上传
- 点击「生成视频」，启动完整 AI 生成流水线

**智能默认：**
- 分镜有台词文案时，即使不手动开启 TTS，系统也会**自动匹配默认音色**生成人声
- 未选择 BGM 时，系统自动从预设库中**智能选取**适合的背景音乐

> 📸 **截图提示 3**：截取板块3 区域，展开「音频设置」面板，可见 TTS 音色选择和 BGM 预设按钮

---

### 2.5 板块 4：视频生成进度与成片展示

**功能描述：**
- 顶部置顶进度条实时显示生成状态（支持每 2 秒轮询，进度平滑过渡动画）
- 生成完成后展示最终成片（含人声 + BGM）
- 同时展示 4 个分镜的独立短视频（每个分镜均含人声 + BGM）
- 支持全屏播放、一键下载、发布到模板库

> 📸 **截图提示 4**：截取板块4 区域，分上下两部分——上方为最终成片的视频播放器，下方为 4 个分镜视频网格

### 2.6 顶部实时进度条

**功能描述：**
- 进度条固定在页面顶部（sticky），滚动页面时始终可见
- 显示当前任务类型、状态、具体步骤（如「正在生成 P2 分镜视频…」）
- 进度值有平滑补间动画，避免长时间静止感
- 支持任务取消

> 📸 **截图提示 5**：截取页面顶部进度条（最好在视频生成中时截图），展示步骤文字和进度百分比

### 2.7 素材库与模板管理

**功能描述：**
- 左侧文件管理器展示已上传的商品图和分镜模板
- 支持拖拽素材到工作台对应板块
- 生成的分镜视频可一键「发布到视频模板库」，供后续项目复用

## 三、技术架构

### 3.1 系统架构图

```
┌─────────────────────────────────────────────────────────────┐
│                        前端（React + Vite）                   │
│  Block1  →  Block2  →  Block3  →  Block4  │  左侧素材库       │
│  商品输入     剧本编辑    分镜配置    视频展示   │  右侧预览                │
└───────────────────────┬─────────────────────────────────────┘
                        │ HTTP / SSE
┌───────────────────────┴─────────────────────────────────────┐
│                      后端（FastAPI）                          │
│                                                             │
│  ┌──────────┐  ┌────────────┐  ┌──────────────────────┐   │
│  │ DirectorAgent │  │ ScriptAgent │  │  VideoGenAgent      │   │
│  │ 任务编排    │  │ 剧本生成    │  │  分镜视频生成         │   │
│  └──────────┘  └────────────┘  └──────────────────────┘   │
│                                                             │
│  ┌──────────────────────────────────────────────────────┐  │
│  │              PostProcessAgent（后处理流水线）           │  │
│  │  TTS 人声合成 → 分镜混音 → BGM → 视频拼接 → 时长适配    │  │
│  └──────────────────────────────────────────────────────┘  │
└──────┬──────────────────┬───────────────────┬──────────────┘
       │                  │                   │
  Seed-2.0-pro      Seedance-1.5-pro     Edge-TTS / FFmpeg
  （剧本生成）        （视频生成）         （语音 + 音视频处理）
```

### 3.2 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端 | React 19 + TypeScript + Vite + Tailwind CSS v4 | 暗黑主题 Studio UI |
| 后端 | FastAPI + Hypercorn + SQLModel + SQLite | 异步 API 服务 |
| AI 剧本 | 火山方舟 Seed-2.0-pro | 多模态理解 + 文案生成 |
| AI 视频 | 火山方舟 Seedance-1.5-pro | 图生视频大模型 |
| 语音合成 | Microsoft Edge-TTS | 中文神经网络语音 |
| 音视频处理 | FFmpeg | 拼接/混音/时长适配 |
| 部署 | Docker + Docker Compose | 一键容器化部署 |

### 3.3 后处理音频流水线

```
Seedance 原始视频（静音）
        ↓  TTS 语音合成（edge-tts）
    每分镜 + 人声配音
        ↓  add_bgm（分镜预览）
    每分镜 + 人声 + BGM         ← Block4 分镜预览带声音 ✓
        ↓  concat_videos（AAC 重编码，流格式对齐）
    完整合并视频（含人声）
        ↓  fit_video_duration（保留音轨，atempo 调速）
    时长适配版本
        ↓  add_bgm（整段连续 BGM）
    最终成片（人声 + 连续无缝 BGM）  ← 完整视频带声音 ✓
```

## 四、核心亮点

### 4.1 全自动 AIDA 营销分镜
系统内置电商营销方法论，AI 生成的分镜严格遵循「注意-兴趣-欲望-行动」四段式结构，确保每条视频都具备营销转化逻辑。

### 4.2 多模态商品理解
剧本生成时同时输入商品图片和文字描述，Seed-2.0-pro 多模态模型能精准识别商品外观特征，生成与商品高度匹配的视觉化文案。

### 4.3 流式实时输出
剧本生成采用 SSE 流式响应，前端使用 ReadableStream 逐字符接收并实时渲染，显著改善用户等待体验。

### 4.4 智能音频流水线
- **自动 TTS**：分镜有台词文案时无需手动开启，系统自动合成人声
- **自动 BGM**：无用户配置时自动选取预设背景音乐
- **音轨对齐**：使用 FFmpeg `ensure_audio_track` 确保所有分镜视频流格式一致，避免 concat 失音
- **速度同步**：时长适配时用 `atempo` 滤镜同步调整音频速度，防止音画不同步

### 4.5 Agent 编排架构
后端采用多 Agent 协作模式：`DirectorAgent` 统一编排任务，`ScriptAgent` 负责剧本，`VideoGenAgent` 负责逐镜生成，`PostProcessAgent` 负责后处理，职责清晰、易于扩展。

### 4.6 容器化一键部署
提供完整的 Dockerfile 和 docker-compose.yml，无需本机安装 Python/Node/FFmpeg，`docker compose up` 即可运行。

## 五、使用流程演示

```
第 1 步：新建项目
  └── 输入商品名称（如「无线蓝牙耳机 Pro」）→ 创建项目

第 2 步：上传商品图（板块1）
  └── 拖拽 1-4 张商品图 → 填写商品描述 → 点击「生成剧本」
  └── 观察剧本文字实时流式出现

第 3 步：生成分镜（板块2）
  └── 确认剧本内容 → 选择视频时长 20 秒 → 点击「生成分镜提示词」
  └── AI 输出 P1~P4 四个分镜的详细画面描述

第 4 步：配置音频（板块3）
  └── （可选）选择 TTS 音色 / 上传 BGM
  └── 点击「生成视频」

第 5 步：等待生成（板块4 + 顶部进度条）
  └── 进度条显示「正在生成 P1 分镜视频…」→「后处理中…」
  └── 约 3~5 分钟后成片出现

第 6 步：预览与下载
  └── 播放最终成片（含人声 + BGM）
  └── 查看 4 个独立分镜视频
  └── 一键下载 / 发布到模板库
```
